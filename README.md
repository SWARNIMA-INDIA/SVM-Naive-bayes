Support Vector Machine (SVM) and Naive Bayes – Machine Learning Lab Assignment

Overview

This repository contains a complete Machine Learning lab assignment focused on the implementation and analysis of Support Vector Machine (SVM) and Naive Bayes algorithms using Python.
The notebook includes 46 practical programs that demonstrate different concepts such as classification, regression, model evaluation, feature selection, cross-validation, and performance visualization.

The objective of this assignment is to understand how SVM and Naive Bayes models work in real datasets and how different evaluation techniques can be used to analyze model performance.

---

Technologies Used

- Python
- Jupyter Notebook
- Scikit-learn
- NumPy
- Matplotlib
- Seaborn

---

Topics Covered

The assignment focuses on the following Machine Learning concepts:

- Support Vector Machine (SVM) Classification
- Support Vector Regression (SVR)
- Naive Bayes Classification
- Kernel Functions
- Feature Scaling
- Feature Selection
- Cross Validation
- Model Evaluation Metrics
- Confusion Matrix Visualization
- ROC Curve and Precision-Recall Curve

---

Datasets Used

The assignment uses built-in datasets from the Scikit-learn library, including:

- Iris Dataset
- Breast Cancer Dataset
- Wine Dataset
- Diabetes Dataset

These datasets are commonly used for learning classification and regression algorithms.

Learning Outcomes

After completing this assignment, the following concepts are clearly understood:

- Working principle of Support Vector Machines
- Implementation of Naive Bayes classifiers
- Importance of feature selection and scaling
- Use of cross-validation techniques

- Visualization of model performance
